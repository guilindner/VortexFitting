GUI_utils module¶

Some functions for the Graphical User Interface

vortexfitting.GUI_utils.gui_plot_accepted(ax: matplotlib.axes.Axes, canvas: matplotlib.backends.backend_agg.FigureCanvasAgg, vfield: VelocityField, vortices_list: list, detection_field: ndarray, flip: bool) → None¶

Plot accepted vortices in GUI: display the accepted vortices with respect to the different criteria.

Parameters:

  • ax (figure) – Matplotlib AxesSubplot for the current plot

  • canvas (canva) – FigureCanvas associated to the axis

  • vfield (class vfield) – spatial grid and velocity components

  • vortices_list (list) – all the detected vortices

  • detection_field (np.ndarray) – detection field (vorticity …)

  • flip (boolean) – invert x and y axes if True.

Returns:

empty

vortexfitting.GUI_utils.gui_plot_fit(ax: matplotlib.axes.Axes, canvas: matplotlib.backends.backend_agg.FigureCanvasAgg, x_index: ndarray, y_index: ndarray, u_data: ndarray, v_data: ndarray, u_model: ndarray, v_model: ndarray, xc: float, yc: float, core_radius: float, gamma: float, u_advection: ndarray, v_advection: ndarray, correlation_value: float, flip: bool) → None¶

Quiver plot of data velocity and model velocity in GUI

Parameters:

  • ax (figure) – Matplotlib AxesSubplot for the current plot

  • canvas (canvas) – FigureCanvas associated to the axis

  • x_index (float matrix) – spatial grid (x direction)

  • y_index (float matrix) – spatial grid (y direction)

  • u_data (float matrix) – data velocity component (u)

  • v_data (float matrix) – data velocity component (v)

  • u_model (float matrix) – model velocity component (u)

  • v_model (float matrix) – model velocity component (v)

  • xc (float) – x coordinate of the vortex center

  • yc (float) – y coordinate of the vortex center

  • core_radius (float) – vortex radius

  • gamma (float) – vortex circulation

  • u_advection (float matrix) – advection velocity component (u)

  • v_advection (float matrix) – advection velocity component (v)

  • correlation_value (float) – corrélation entre le vortex et un modèle Lamb-Oseen.

  • flip (boolean) – invert x and y axes if True.

Returns:

empty

Return type:

empty
