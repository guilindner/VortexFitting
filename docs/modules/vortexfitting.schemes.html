Schemes module¶

Difference schemes for the velocity derivatives (updated with SciPy convolutions).

Tip

new difference schemes should be added here

vortexfitting.schemes.fourth_order_diff(vfield: VelocityField) → ndarray¶

Fourth order accurate finite difference scheme using SciPy convolutions.

Note

Kernel: [1, -8, 0, 8, -1] / (12 dx)

Params vfield:

2D array of the velocity field, containing u and v

Returns:

derivative

Return type:

2D array of float

vortexfitting.schemes.least_square_diff(vfield: VelocityField) → ndarray¶

Least-square 5-point derivative scheme via SciPy convolution.

Note

Kernel (Raffel 1998): [-2, -1, 0, 1, 2] / (10 dx)

Params vfield:

2D array of the velocity field, containing u and v

Returns:

derivative

Return type:

2D array of float

vortexfitting.schemes.second_order_diff(vfield: VelocityField) → ndarray¶

Second order accurate finite difference scheme using SciPy convolutions.

Note

Kernel: [-1, 0, 1] / (2dx)

Params vfield:

2D array of the velocity field, containing u and v

Returns:

derivative

Return type:

2D array of float
