# Turbulent Boundary Layer PIV Dataset – Quick Start Guide

This dataset comprises experimental measurements of a zero-pressure-gradient flat-plate boundary layer at five different Reynolds numbers, collected using planar particle image velocimetry (PIV). Each dataset includes approximately 6,000 snapshots of planar velocity fields and raw particle image pairs, stored in HDF5 format. A MATLAB script is provided to facilitate data reading and manipulation.

## Getting Started

To quickly test the [VortexFitting](https://gitlab.com/ydevaux/vortexfitting) software, download the `BLexp_example.zip` file from the dataset repository:

[Dataset Repository](https://deepblue.lib.umich.edu/data/concern/data_sets/gq67jr57z?locale=en)

[Download BLexp_example.zip - 3.5Gb](https://deepblue.lib.umich.edu/data/downloads/fq977v74z)

This example file contains a subset of the full dataset, suitable for initial testing and familiarization with the VortexFitting tool.
